## Copy

The `Copy` command copies the `N`th item on the stack onto the top of the stack. It was added in Whitespace v0.3<sup><a href="#sup1" style="text-decoration: none">1</a></sup>.

### Syntax

<SyntaxTable command="Copy" inter="sts" />

### Parameters

- `Int` - the index from the top of the stack to copy the item from.

### Stack parameters

The `Copy` command consumes no stack parameters, however will copy the `N`th item as per the parameter provided.

### Example

<Example client:load="react" example={`Push 1
Push 2
Push 3
Copy 1`}/>


## Slide

The `Slide` command keeps the top item of the stack, but removes `N` items beneth it. It was added in Whitespace v0.3<sup><a href="#sup1" style="text-decoration: none">1</a></sup>.

### Syntax

<SyntaxTable command="Slide" inter="stn" />

### Parameters

- `Int` - the number of items to discard off the stack.

### Stack parameters

The `Slide` command will remove `N` items beneth the top item on the stack, however it's behavior will not change based upon the values removed.

### Example

<Example client:load="react" example={`Push 1
Push 2
Push 3
Slide 2`}/>
