## ReadChar

The `ReadChar` command reads a character from the input and writes it to a provided heap address.

### Syntax

<SyntaxTable command="ReadChar" />

### Parameters

The `ReadChar` command takes no parameters

### Stack parameters

- `Int` - the heap address to write the character to.
